Абдулов Даннил 211-324

Здесь будут расписаны различные адресные запросы и методы, которые их обрабатывают:


  • "/" - Корневой путь
    На Get - запрос, метод: rootHandler

    Краткое описание: Метод отправляет ответ в виде сообщения: "Hello world!"

  • "/comments" - Запрос на отправку/получение комментариев
    На Post - запрос, метод: commentsHandler

    Краткое описание: Метод может принимать в себя тело запроса, после этого изменяя массив комментариев, в конце, отправляет текущий список комментариев

  • "/stats" - Корневой путь
    На Get - запрос, метод: statsHandler

    Краткое описание: Метод обрабатывает запрос на получение статуса, в заранее заготовленный объект добавляется информация о посетители сайта и количестве его запросов, затем в виде html - таблицы отправляется в виде ответа пользователю

  • Остальные запросы
    Метод: allHandler

    Краткое описание: Метод обрабатывает остальные запросы, ответ в виде статуса 400, Bad Request
